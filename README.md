# ESS-PCB-for-Formula-Student-Oulu
ESS or Extra Sensory System for Fromula Student. Project was done with Kicad and later on imported to EasyEda. Chosen sensor for the PCB was XGZP6847A. PCB also has 12V to 5V regulator and two mounting holes. I also had to add sponsors logos to the PCB.
